## 2. 시스템 아키텍처 및 기술 스택

### 2-1. Core Backend
- **엔진**: Node.js / Express.js
- **데이터베이스**: **MySQL (InnoDB)** - 고성능 데이터 처리 및 관리
- **인증 시스템**: JWT 기반 보안 토큰 관리 & Bcrypt 비밀번호 암호화
- **인프라**: 자동 스키마 초기화 로직 및 트랜잭션 무결성 보장

### 2-2. Core Frontend
- **기반**: Vanilla JavaScript (ES6+), Semantic HTML5
- **스타일링**: CSS Variables 기반 전역 테마 시스템 (세련된 무채색 & 웜 카라멜 액센트)
- **에셋**: 고해상도 이미지 (3:4 비율) 및 최적화된 SVG 아이콘 시스템

---

## 3. 핵심 비즈니스 로직 및 기능

### 3-1. 멤버십 체계 (6단계)
사용자의 기여도와 구매력에 따른 세분화된 혜택을 제공합니다.
- **고객 등급**: `bronze` (웰컴), `silver`, `gold`, `dia` (프리미엄)
- **운영 등급**: `매니저`, `관리자` (시스템 제어 및 전권한)

### 3-2. 주문 및 결제 자동화
- **트랜잭션**: 재고-결제-포인트가 하나로 묶인 원자적 처리 프로세스.
- **비회원 프로세스**: 20~30 세대의 편의성을 고려한 간편 비회원 주문 및 사후 조회 시스템.

### 3-3. 보안 및 신뢰성
- **개인정보 보호**: 결제 정보 대리 저장 금지 및 실시간 세션 보호.
- **데이터 정비**: 24시간 주기 비회원 임시 데이터 자동 청소 스케줄링.

---

## 4. 리스크 관리 및 향후 고도화 제안

### 4-1. 리스크 대응 방안
- **데이터베이스 복원력**: MySQL 서버 순단 시 자동 재연결 로직 및 에러 핸들링 고도화.
- **인증 보안 강화**: 무차별 대입 공격 방지를 위한 로그인 시도 제한(Rate Limiting) 도입 권고.
- **리소스 최적화**: 고해상도 패션 이미지의 효율적 로딩을 위한 WebP 전환 및 CDN 활용 검토.

### 4-2. 미래 지향적 개선 (Future Roadmap)
- **통합 관리자 센터**: 실시간 매출 분석 및 상품 배포가 가능한 고도화된 Admin UI 구축.
- **AI 스타일링 추천**: 20~30 고객의 취향을 분석하여 상품을 제안하는 큐레이션 알고리즘 도입.
- **모바일 앱 확장**: PWA 및 하이브리드 앱 전환을 통한 푸시 알림 및 접근성 강화.
