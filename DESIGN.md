## 2. 시스템 아키텍처 및 기술 스택

### 2-1. Core Backend
- **엔진**: Node.js / Express.js
- **데이터베이스**: **MySQL (InnoDB)** - 고성능 데이터 처리 및 관리
- **인증 시스템**: JWT 기반 보안 토큰 관리 & Bcrypt 비밀번호 암호화
- **하이브리드 확장**: 카카오·네이버·구글 소셜 로그인 시뮬레이션 인터페이스 구축
- **인프라**: 자동 스키마 초기화 로직 및 24시간 주기 만료 데이터(비회원 장바구니/위시리스트) 자동 정리

### 2-2. Core Frontend
- **기반**: Vanilla JavaScript (ES6+), Semantic HTML5
- **스타일링**: CSS Variables 기반 전역 테마 시스템 (무채색 & 웜 카라멜 액센트)
- **레이아웃**: Editorial 1:8 비대칭 그리드 시스템 (Weekly Best 영감)
- **에셋**: 순수 **SVG 아이콘 시스템** 도입 (레거시 이미지 의존성 제거, 성능 최적화)

---

## 3. 핵심 비즈니스 로직 및 기능

### 3-1. 멤버십 및 인증 체계
- **고객 등급**: `bronze` (웰컴), `silver`, `gold`, `dia` (프리미엄)
- **보안 기능**: 프리미엄 토글 스위치 방식의 '로그인 상태 유지', 실시간 세션 보호
- **소셜 연동**: 브랜드 일관성을 유지하는 커스텀 소셜 로그인 모달 시스템

### 3-2. 고도화된 관리자 대시보드 (Admin)
- **상품 관리**: 다중 선택 가능한 상품 배지 시스템 (`NEW`, `MD추천`, `주문폭주`, `SALE`)
- **운영 편의성**: 브라우저 새로고침 시에도 유지되는 페이지 상태 보존(State Persistence) 기능
- **카테고리 제어**: 직관적인 계층형 카테고리 관리 및 순서 재배치

### 3-3. 주문 및 결제 프로세스
- **트랜잭션 관리**: 재고-결제-포인트 일괄 원자적 처리
- **비회원 서비스**: 비밀번호 보호 기반의 간편 비회원 주문 및 사후 조회 시스템

---

## 4. 리스크 관리 및 향후 고도화

### 4-1. 최적화 및 보안
- **성능**: Gzip 압축 및 캐싱 최적화를 통한 로딩 속도 극대화
- **보안**: Prepared Statements를 통한 SQL Injection 원천 차단

### 4-2. 미래 지향적 개선 (Future Roadmap)
- **통합 정산 시스템**: 실시간 매출 분석 및 파트너 정산 기능 고도화
- **AI 큐레이션**: 사용자 활동 패턴 분석 기반의 자동 스타일 제안 알고리즘 도입
- **확장성**: PWA 기반 모바일 앱 서비스 확장을 통한 접근성 강화
